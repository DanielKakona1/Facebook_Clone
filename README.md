# Facebook_Clone

Facebook_Clone is a clone of Facebook.

## 🚀 Techs

- ⚛️ Reactjs

## ✋🏻 Prerequisites

- [Node.js](https://nodejs.org/)

## 🔥 Install & Execute

1. Clone the repo;
2. Run `cd facebook_Clone` to open the cloned repo;
3. Run `yarn install` to install the dependecies;
4. Run `yarn start` to run the system;
